# Airdrop

A snapshot of the Starknet mainnet state was taken on **March 26, 2026**. Eligible controllers can claim their free games directly on the game website. A controller may qualify under multiple categories—rewards stack.

| Criteria                                     | Free games                                             |
| -------------------------------------------- | ------------------------------------------------------ |
| Cartridge Controller created before snapshot | **1**                                                  |
| Owned at least 1 EGS NFT before snapshot     | **1**                                                  |
| Legacy NUMS balance at snapshot              | < 5 000: **1**<br/>< 50 000: **2**<br/>≥ 50 000: **3** |

:::tip
**Stacking.** Rewards from different categories stack. A controller created before the snapshot that also held 5 000 legacy NUMS tokens would receive 1 + 2 = 3 free games.
:::

:::warning
All airdrop claims must be made before **July 1, 2026 at 00:00 UTC**. Unclaimed free games after this date are forfeited.
:::

## Cartridge Controller

- All Cartridge Controllers created before the snapshot date are eligible.

## Embedded Game Standard (EGS) NFT

- Controllers that owned at least one Embedded Game Standard NFT before the snapshot are eligible.
- EGS is a token standard developed by [Provable Games](https://docs.provable.games/embeddable-game-standard) on Starknet mainnet.

## Legacy NUMS token

- Users who held the original NUMS token—used as a reward during the first version of the game—are eligible based on their balance at the time of the snapshot.
- The legacy NUMS token is distinct from the current NUMS token. Only balances of the original reward token are considered.
